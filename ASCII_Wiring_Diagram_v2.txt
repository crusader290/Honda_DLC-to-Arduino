Honda 3-Pin DLC (OBD1)
                  ----------------------
 Pin1: K-Line  ----+----------------------------------+
                   |                                  |
                   |                                [1kΩ]
                   |                                  |
                   |                                 Nano RX (D3)
                   |                                  |
                   |                               Schottky Diode
                   |                        Anode → Nano RX
                   |                        Cathode → DLC Pin1
                   |                                  |
                   |                               47µF Capacitor
                   |                                  |
                   +--------------------------------> GND rail
                   
 Pin2: +12V  -----[0.5A Fuse]----> Buck IN (+12V)
                                     |
                                     +--> Buck OUT 5V --> Nano VCC
                                     +--> Buck OUT 5V --> OLED VCC
 Pin3: GND  -------------------------> GND rail
                                     +--> Nano GND
                                     +--> OLED GND

Nano TX (D2) -------------------------> K-Line TX (optional small resistor to DLC Pin1)

OLED:
  SDA ---------------------------> Nano A4
  SCL ---------------------------> Nano A5
  VCC --------------------------> Buck 5V OUT
  GND --------------------------> GND rail

Protection Component Notes:
- **1kΩ series resistor**: limits current to Nano RX
- **Schottky diode**: clamps voltage, protects RX (Anode → Nano RX, Cathode → DLC K-Line)
- **47µF capacitor**: smooths K-Line voltage transients
- **0.5A fuse**: protects 12V supply to buck and DLC Pin2
- **Buck converter**: powers Nano and OLED safely from 12V car battery

Summary:
- All power comes from DLC Pin2 (+12V)
- GND from DLC Pin3
- K-Line data from DLC Pin1, protected with resistor + diode + capacitor
- Nano TX/RX handle bidirectional K-Line communication
- OLED displays live RPM, Fuel %, Coolant temp
- USB Serial logs all live data
- EEPROM stores last readings; no SD or Bluetooth needed
